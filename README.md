# Passkeys POC with NextJS

## Getting Started

### Install packages

please run:

```sh
npm install
```

### Initialize the Database

- copy the .env.local to .env ( create it please )
- then run this to populate the database based on `prisma/schema.prisma`

```sh
npx prisma db push
```

### Run the project

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
